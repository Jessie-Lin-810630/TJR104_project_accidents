# 本機安裝redis
1. 安裝 
    ```
    brew install redis # 約5 MB
    ```

2. 啟動服務
    ```
    brew service start redis # 跳出Successfully started `redis` (label: homebrew.mxcl.redis)
    ```
3. 查看服務狀態
    ```
    brew services list
    ```
4. 啟動並進入伺服器
    ```
    redis-server
    ```
5. [optional]另開terminal，進入cli
    ```
    redis-cli

    #基礎檢查指令
    ping # 回傳pong
    set foo bar # 回傳OK
    get foo # 回傳"bar"
    ```
6. [optional]離開cli
    ```
    # ctrl+C / command+C
    ```
6. 停止服務
    ```
    brew services stop redis 或 啟動redis-server的終端機界面按下ctrl+C / command+C
    ```

# python -reids套件
1. 建立連線池
    ```
        # 好處：後續平行運算取用時能減少重複建立連線的開銷，避免佔用過多系統資源
        redis_pool = redis.ConnectionPool(host=host, port=port, 
                            password=password if password else None, 
                            decode_responses=False)
    ```
2. 建立連線物件client
    ```
        r = redis.Redis(connection_pool=redis_pool)
    ```
3. 存入二進位資料
    ```
        import pickle
        values = pd.DataFrame(.....).to_dict("records")
        pickled_data = pickle.dumps(values) # 壓縮成二進位

        # r from step 1~2.
        # key: a str like the namecard of stored pickled-data in Redis.
        # ttl: time to live. Unit: second
        r.setex(key, ttl, pickled_data)
    ```
