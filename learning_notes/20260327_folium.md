# c_ui.py
1. Folium 是一個基於 Leaflet.js 的 Python 套件，可用來建立互動式地圖。與Pandas、GeoPandas整合：可將地理數據（CSV、Shapefile、GeoJSON）視覺化。也可匯出 HTML，嵌入 Flask/Django 應用。適合用於 “地理數據可視化與空間分析”：
    (1) 在地圖上標記特定地點 (如餐廳、景點、店鋪位置)。 <-- marker
    (2) 畫出商圈範圍、服務範圍、城市行政區、國家"邊界"，並顯示人口或疫情數據。 <-- circle
    (3) 分析人流密度，如商業區、交通流量、犯罪數據。 <-- HeatMap、MarketCluster
    (4) 與Pandas整合，將 Excel/csv數據轉換為地圖。

2. folium.Map(): 初始化地圖，回傳地圖物件
    ```
        m = folium.Map(location=loc, zoom_start=zoom, tiles=custom_tiles, prefer_canvas=True)
        # location: 地圖中心經緯度。需為list或tuple，緯度在前、經度在後。
        # zoom_start: 初始比例尺
        # titles: 地圖背景樣式。常見有OpenStreetMap、CartoDB positron、Stamen Terrain等。
        # prefer_canvas: prefer_canvas=True可強制 Folium 使用 HTML5 Canvas 繪製點位，大幅提升繪製幾千個點的效能。
    ```
3. 儲存成 HTML
    ```
    m.save("basic_map.html")
    ```

4. 兩種能表達數據密度的圖層，HeatMap()、MarkerCluster()。這兩種都是做出一層圖層。
    (1)熱力圖（HeatMap）：適合顯示數據密度，例如 人流量、氣候數據、房價分布。
        ```
            HeatMap(data, radius, blur, min_opacity).add_to(m)
            # data: a list of [lat, long] 或 a list of (lat, long)。

            # 從pandas dataframe轉成list傳入data參數：可以先使用df[ [lat, long] ].values.tolist()得到列表後傳入data。

            # radius: int，表達一個點的像素大小，預設25。值越大，數據點的擴散面積越大，因此當所有點都畫上去之後，也會讓熱區面積變大。

            # blur: int，表達點的模糊程度，預設15，小則邊界清楚、大則柔和。

            # min_opacity: 0~1的小數點，表示最淡點位的最低透明度，預設通常是0，可以是否過濾掉弱訊號強度。
            0代表幾乎忽略弱訊號(低密度)。通常可選0.3。

            # m: 用 folium.Map() 建立的地圖物件變數名稱。
        ```

    (2)點群集（MarkerCluster）搭配CircleMarker()：適合 大量標記點的地圖視覺化，例如 餐廳、停車場、災害報告。
        ```
            # Step 1.製作Cluster容器：把多個 marker 集中管理，並在縮放時自動群聚（cluster）
                MarketCluster(data, disableClusteringAtZoom).add_to(m)
                # data: a list of [lat, long]
                # disableClusteringAtZoom=16: 當地圖放大到 level 16 時，強制散開所有群聚點位以利檢視
                # m: 用 folium.Map() 建立的地圖物件變數名稱

            # Step 2.繪製點並放入容器
                # 使用CircleMarker()或Marker()
                folium.Marker([target_market['lat'], target_market['lon']], 
                            icon=folium.Icon(color='purple', icon='star', prefix='fa'), tooltip=target_market['MarketName']).add_to (fg_m)

                folium.Marker([target_market['lat'], target_market['lon']],
                          icon=folium.Icon(color='purple', icon='star', prefix='fa'),  
                          tooltip=target_market['MarketName']).add_to(fg_m)

                folium.CircleMarker([r.latitude, r.longitude],
                                    radius=5, color=color, fill=True, fill_opacity=0.7,
                                    popup=folium.Popup(popup_text, max_width=200)
                                    ).add_to(cluster_other)

                


        ```
5. 如果想在一張地圖同時放多個圖層、並允許使用者點選要看哪一個圖層，使用FeatureGroup()
    ```
        # Step1. 建立群組fg_m
        fg_m = folium.FeatureGroup(name="夜市周圍邊界")
        
        # Step2. 個別繪製每層圖層，然後加入群組fg_m。
            # 範例1:
            # radius: 半徑(m) # color: 邊框顏色 # fill=True: 填滿圓內部 # fill_opacity: 填色時的透明度
            folium.Circle([target_market['lat'], target_market['lon']], radius=radius_m, color='orange', 
            fill=True, fill_opacity=0.1).add_to(fg_m)

            # 範例2:
            folium.Marker([target_market['lat'], target_market['lon']], icon=folium.Icon(color='purple', icon='star', prefix='fa'), tooltip=target_market['MarketName']).add_to(fg_m)

        # Step3. 再把圖層群組fg_m加到地圖物件中。
            fg_m.add_to(m)

        # Step4. 加上圖層控制面板，讓使用者有地方點選
            folium.LayerControl(collapsed=False).add_to(m) # 預設會在右上角呈現。
    ```
